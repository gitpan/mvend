MiniVend Tech Tip

MiniVend supports SSL security, which allows secure input of sensitive information like credit cards. It is relatively simple to use -- this demo has it enabled by default. If you have the demo up, but no secure server, edit the SecureURL directive with MAT (or by changing minivend.cfg) and just use http instead of https for the protocol specification. See the file doc/minivend.html for more information.
